## 💡 주요 기능

- 가게 OPEN 및 CLOSE 후 데이터 마감정산
- 카테고리 & 메뉴 SelfJoin이용
- 각 테이블에 들어간 주문 상황을 실시간으로 시각적으로 보여주는 기능
- 테이블의 이동, 테이블삭제, 합석 기능 구현
- 결제 프로세스와 당일 매출금액 및 차트 확인기능구현
